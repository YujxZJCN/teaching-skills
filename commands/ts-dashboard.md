description: TS teaching-pipeline `dashboard` mode — single-file HTML course overview
---

Trigger the `teaching-pipeline` skill in `dashboard` mode. Runs `python3 scripts/build_dashboard.py <passport>` to generate `course_dashboard.html` next to the Course Passport: gates with a live re-check, outcomes, assessment plan, week-by-week resources with clickable local artifact links, the artifacts ledger, workload, and iteration history. Open it in any browser; regenerate whenever the passport changes.

Mode reference: `MODE_REGISTRY.md`. Skill entry: `teaching-pipeline/SKILL.md`.
