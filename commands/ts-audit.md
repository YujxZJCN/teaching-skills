Trigger the `submission-auditor` skill in `audit` mode (or `batch-audit` for multiple submissions). Audits student work against the confirmed Submission Spec with location-cited findings, runs the batch fairness pass, and produces per-student feedback report drafts plus a class-level pattern report.

Mode reference: `MODE_REGISTRY.md`. Skill entry: `submission-auditor/SKILL.md`.
